# Rodada 02 - Documentation Review

## Objetivo

Padronizar linguagem, títulos, índice, links, exemplos, checklists e consistência documental.

## Entradas

Documentos alterados, `INDEX.md`, templates, links internos e padrão de documentação.

## Critérios

- Português do Brasil.
- Estrutura consistente.
- Links funcionais.
- Conteúdo acionável.
- Ausência de repetição inútil.

## Checklist

- [ ] Títulos seguem padrão.
- [ ] Índice foi atualizado.
- [ ] Links foram verificados.
- [ ] Exemplos existem quando úteis.
- [ ] Conteúdo genérico foi removido.

## Saída esperada

Lista de ajustes documentais e arquivos que exigem atualização.

## Conclusão

Documentação boa reduz ambiguidade operacional.
