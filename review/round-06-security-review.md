# Rodada 06 - Security Review

## Objetivo

Avaliar riscos de segurança, privacidade, exposição de dados, segredos, permissões e abuso previsível.

## Entradas

Mudança proposta, dados envolvidos, integrações, permissões, logs, uploads, webhooks e dependências externas.

## Critérios

- Autenticação.
- Autorização.
- Dados sensíveis.
- Segredos.
- Validação de entrada.
- Auditoria.

## Checklist

- [ ] Permissões foram avaliadas.
- [ ] Dados sensíveis foram minimizados.
- [ ] Segredos não foram expostos.
- [ ] Entradas externas foram validadas.
- [ ] Logs são seguros.

## Saída esperada

Parecer de risco, bloqueios críticos e controles necessários.

## Conclusão

Falha crítica de segurança bloqueia entrega.
