# Best Dashboard

## Objetivo

Registrar padrões aprovados para dashboards operacionais, executivos ou analíticos.

## Critérios

- Indicadores acionáveis.
- Exceções visíveis.
- Filtros úteis.
- Comparações claras.
- Gráficos com propósito.

## Quando promover para conhecimento

Promova para o Core apenas quando o padrão for genérico, sem dados privados e aplicável a múltiplos produtos.
