# Best Layouts

## Objetivo

Registrar padrões de composição aprovados para tipos recorrentes de tela.

## Critérios para entrada

- Resolveu tarefa real.
- Obteve score visual alto.
- Pode ser reutilizado com adaptação.
- Não contém dado sensível.
- Possui justificativa de layout.
