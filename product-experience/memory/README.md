# Experience Memory Catalog

## Objetivo

Organizar tipos de referência que podem ser registrados em Workspaces e, quando forem genéricos, promovidos para conhecimento reutilizável no Core.

## Catálogo

- `best-layouts.md`
- `best-dashboard.md`
- `best-table.md`
- `best-form.md`
- `best-sidebar.md`

## Regra

Memória de experiência deve explicar decisão, contexto e limite de reutilização. Não deve armazenar dados sensíveis nem copiar telas de clientes ou produtos externos.
