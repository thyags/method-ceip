# Best Table

## Objetivo

Registrar padrões aprovados para tabelas operacionais e listas densas.

## Critérios

- Colunas orientadas por decisão.
- Densidade adequada.
- Ações contextuais.
- Status acessível.
- Filtros e ordenação úteis.

## Quando evitar reutilização

Evite quando o usuário precisa entender cada item como narrativa, quando a comparação não importa ou quando cards aumentam clareza.
