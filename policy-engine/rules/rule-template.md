# Rule Template

## Objetivo

Padronizar criação de regras do Policy Engine.

## Contexto

Regras operacionalizam políticas. Elas devem ser claras para humanos e IAs.

## Template

```markdown
# Regra - Nome

## Objetivo

Que decisão esta regra governa.

## Contexto

Quando a regra se aplica.

## Gatilho

Condição que ativa a regra.

## Diretriz

O que deve acontecer.

## Exceções

Quando a regra pode ser flexibilizada.

## Evidência

O que comprova cumprimento.

## Checklist

- [ ]

## Conclusão

Resultado esperado.
```

## Exemplos

Use este template para criar regra de aprovação, ADR, documentação ou roteamento.

## Checklist

- [ ] Gatilho está claro.
- [ ] Diretriz é verificável.
- [ ] Exceções foram descritas.
- [ ] Evidência foi definida.

## Conclusão

Regras consistentes tornam o Policy Engine executável.
