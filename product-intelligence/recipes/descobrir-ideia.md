# Recipe: Descobrir Ideia

## Quando usar

Quando o usuário traz apenas uma frase inicial.

## Passos

1. Registrar a frase original.
2. Perguntar problema, usuário, impacto e urgência.
3. Separar solução imaginada de dor real.
4. Criar Idea Brief.
5. Encaminhar para Discovery Engine.

## Saída

- `idea-template.md` preenchido.
- Lacunas para discovery.

## Gate

- Não avançar para PRD sem problema e usuário claros.
