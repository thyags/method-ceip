# Templates do Product Intelligence System

## Objetivo

Centralizar templates reutilizáveis para transformar discovery em artefatos de produto.

## Uso

Copie o template necessário para o projeto consumidor, preferencialmente dentro de `.ceip/artifacts/specs/`, e preencha com evidências do projeto.
