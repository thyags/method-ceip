# PRD Quality Checklist

## Estrutura

- [ ] Resumo executivo.
- [ ] Problema.
- [ ] Objetivos.
- [ ] Público-alvo.
- [ ] Personas.
- [ ] Stakeholders.
- [ ] Escopo.
- [ ] Fora do escopo.
- [ ] RF e RNF.
- [ ] Regras de negócio.
- [ ] Fluxos e casos de uso.
- [ ] Integrações.
- [ ] Segurança, performance e UX.
- [ ] Critérios de aceite.
- [ ] KPIs e métricas.
- [ ] Dependências, premissas e restrições.
- [ ] Riscos.
- [ ] MVP.
- [ ] Roadmap.
- [ ] Versões futuras.

## Qualidade

- [ ] Não há requisito sem origem.
- [ ] Lacunas estão explícitas.
- [ ] Critérios são testáveis.
- [ ] MVP é menor que a visão completa.
- [ ] Handoff para arquitetura está claro.
