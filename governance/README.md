# Governance

## Objetivo

Reunir documentos de governança da CEIP e orientar decisões que cruzam domínios.

## Documentos relacionados

- `CONSTITUTION.md`
- `MANIFESTO.md`
- `POLICY_ENGINE.md`
- `ORCHESTRATOR.md`
- `QUALITY_STANDARD.md`
- `DECISION_FRAMEWORK.md`
- `ROADMAP.md`

## Regras

- Mudanças estruturais exigem atualização de índice.
- Decisões arquiteturais relevantes exigem ADR.
- Mudanças de escopo amplo exigem RFC.
- Exceções a gates exigem aprovação formal.

## Checklist

- [ ] A mudança respeita a constituição.
- [ ] Policies foram aplicadas.
- [ ] Gates foram considerados.
- [ ] ADR ou RFC foi criado quando necessário.

## Conclusão

Governança mantém a CEIP coerente enquanto evolui.
