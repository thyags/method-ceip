# Examples

## Objetivo

Reunir exemplos operacionais de uso da CEIP.

## Exemplos esperados

- Classificação de demanda.
- Roteamento de agentes.
- Aplicação de policy.
- Execução de quality gate.
- Registro de ADR ou RFC.
- Resultado de projeto piloto.

## Checklist

- [ ] Exemplo usa documentos reais da CEIP.
- [ ] Exemplo é agnóstico de tecnologia.
- [ ] Saídas são verificáveis.
- [ ] Riscos e limites são declarados.

## Conclusão

Exemplos ajudam adoção quando mostram uso real, não teoria isolada.
