# RFCs Rejeitadas

## Objetivo

Armazenar RFCs rejeitadas para preservar contexto e evitar retrabalho.

## Critérios

- Alternativa recusada.
- Justificativa documentada.
- Condições para reabrir descritas quando aplicável.

## Checklist

- [ ] Motivo da rejeição foi registrado.
- [ ] Alternativa escolhida foi indicada.
- [ ] Condições futuras foram descritas.

## Conclusão

RFC rejeitada também é aprendizado.
