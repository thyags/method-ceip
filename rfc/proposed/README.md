# RFCs Propostas

## Objetivo

Armazenar RFCs ainda em discussão.

## Critérios

- Mudança ampla.
- Impacto em múltiplos módulos.
- Necessidade de consenso antes de implementação.

## Checklist

- [ ] Problema está claro.
- [ ] Alternativas foram listadas.
- [ ] Impactos foram avaliados.
- [ ] Decisão ainda não foi aceita.

## Conclusão

RFC proposta não é decisão final.
