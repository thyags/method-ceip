# Prompt de Projeto Piloto

## Objetivo

Testar a CEIP em um projeto real, preferencialmente GSA Oficina.

## Prompt

```text
Utilize exclusivamente a CloudSix Engineering Intelligence Platform para analisar este projeto piloto.

Não altere produção.
Não invente regra de negócio.
Não assuma stack sem inspeção.

Execute:
- leitura de README, INDEX, CONSTITUTION, ORCHESTRATOR e POLICY_ENGINE;
- classificação de projeto, risco e domínios;
- roteamento de agentes;
- aplicação de policies e quality gates;
- identificação de lacunas do framework;
- recomendações de melhoria para CEIP.

Entregue relatório com achados, documentos insuficientes, ajustes sugeridos e próximos passos.
```

## Checklist

- [ ] Projeto foi analisado com CEIP.
- [ ] Lacunas foram identificadas.
- [ ] Recomendações foram registradas.
- [ ] Roadmap foi atualizado quando necessário.

## Conclusão

Piloto valida a plataforma contra realidade operacional.
