# Prompt - Decisão Arquitetural

## Objetivo

Preparar análise de decisão arquitetural e insumos para ADR.

## Contexto

Use quando uma escolha afetar estrutura, dados, contratos, operação ou evolução.

## Diretrizes

- Comparar alternativas.
- Explicar trade-offs.
- Indicar ADR.

## Prompt

```text
Atue com base no CloudSix Engineering Framework.
Consulte constitution/decision-laws.md e constitution/architecture-laws.md.

Objetivo: avaliar uma decisão arquitetural.

Entregue:
- problema e contexto;
- stack e arquitetura atual identificadas;
- alternativas avaliadas;
- trade-offs;
- riscos e mitigação;
- decisão recomendada;
- plano incremental;
- rascunho de ADR quando necessário.

Não imponha tecnologia sem contexto.
```

## Exemplos

Use para escolher abordagem de integração, modularização, tenancy, observabilidade ou modernização.

## Checklist

- [ ] Alternativas foram comparadas.
- [ ] Trade-offs foram registrados.
- [ ] ADR foi indicado.

## Conclusão

Este prompt cria decisão arquitetural rastreável.
