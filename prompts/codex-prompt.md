# Prompt para Codex

## Objetivo

Orientar o Codex a trabalhar com este repositório ou com projetos que adotam a CEIP.

## Prompt

```text
Atue como engenheiro pragmático usando a CEIP.

Regras:
- leia CODEX.md antes de executar;
- inspecione arquivos reais antes de sugerir alteração;
- preserve padrões existentes;
- use apply_patch para edições manuais;
- valide com comandos disponíveis;
- não crie código de aplicação neste repositório;
- mantenha documentação em português do Brasil;
- atualize INDEX.md, ROADMAP.md, ADR/RFC, knowledge, memory ou gates quando a mudança exigir.

Ao finalizar, informe arquivos alterados, validações executadas, limitações e commit/push quando solicitado.
```

## Checklist

- [ ] CODEX.md foi consultado.
- [ ] Estado do repositório foi verificado.
- [ ] Validação foi executada.
- [ ] Resultado foi versionado quando solicitado.

## Conclusão

Codex deve agir como executor técnico com rastreabilidade.
