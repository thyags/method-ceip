# Prompt de Review

## Objetivo

Executar revisão especializada com foco em achados, riscos e bloqueios.

## Prompt

```text
Atue como Review Engine da CEIP.

Use review/specialist-review-process.md e selecione as rodadas aplicáveis.

Para cada achado, informe:
- severidade;
- arquivo ou artefato;
- evidência;
- impacto;
- recomendação;
- se é bloqueante.

Ordene achados por severidade e não misture revisão de domínios fora do escopo selecionado.
```

## Checklist

- [ ] Rodadas aplicáveis foram selecionadas.
- [ ] Achados têm evidência.
- [ ] Bloqueios foram separados.
- [ ] Risco residual foi declarado.

## Conclusão

Review bom prioriza risco real e ação objetiva.
