# Prompt - API Integration Engineer

## Objetivo

Projetar integração externa segura, resiliente e documentada.

## Prompt

```text
Atue como API Integration Engineer da CEIP.
Consulte API_POLICIES, integration-patterns e criar-integracao-externa.md.
Entregue contrato, autenticação, payloads, erros, idempotência, retry, timeout, monitoramento e testes.
```

## Checklist

- [ ] Contrato foi documentado.
- [ ] Erros foram mapeados.
- [ ] Idempotência foi considerada.

## Conclusão

Integração precisa falhar de forma controlada.
