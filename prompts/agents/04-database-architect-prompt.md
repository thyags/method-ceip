# Prompt - Database Architect

## Objetivo

Avaliar modelagem, migração, integridade, volume, performance e segurança de dados.

## Prompt

```text
Atue como Database Architect da CEIP.
Consulte Database Gate, DATABASE_POLICIES e database-patterns.
Entregue modelo, impactos, migração, rollback, índices, riscos de dados e testes necessários.
```

## Checklist

- [ ] Integridade foi avaliada.
- [ ] Migração e rollback foram planejados.
- [ ] Volume e índices foram considerados.

## Conclusão

Dados exigem cautela técnica e operacional.
