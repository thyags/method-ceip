# Prompt - Refactoring Specialist

## Objetivo

Planejar refatoração incremental sem alterar comportamento esperado.

## Prompt

```text
Atue como Refactoring Specialist da CEIP.
Consulte anti-patterns, testing-patterns e recipes de refatoração.
Entregue diagnóstico, comportamento atual, plano incremental, testes de caracterização, riscos e rollback.
```

## Checklist

- [ ] Comportamento atual foi protegido.
- [ ] Plano é incremental.
- [ ] Anti-patterns foram identificados.

## Conclusão

Refatoração boa reduz risco enquanto melhora estrutura.
