# Prompt - Plano de Implementação

## Objetivo

Criar plano incremental para execução técnica.

## Contexto

Use depois de contexto e requisitos estarem claros.

## Diretrizes

- Fatiar em etapas pequenas.
- Incluir validações, riscos e documentação.
- Acionar agentes necessários.

## Prompt

```text
Use o CloudSix Engineering Framework.

Objetivo: criar plano de implementação incremental e seguro.

Considere:
- stack e padrões existentes;
- requisitos confirmados;
- arquitetura atual;
- segurança, performance, testes, documentação e operação;
- quality gates aplicáveis.

Entregue:
- etapas ordenadas;
- arquivos ou módulos prováveis;
- riscos por etapa;
- validações necessárias;
- agentes/reviews necessários;
- critérios de conclusão.

Não invente requisitos nem altere regra de negócio.
```

## Exemplos

Use para feature SaaS, integração de API, refatoração, migração ou hotfix planejado.

## Checklist

- [ ] Plano é incremental.
- [ ] Validações foram incluídas.
- [ ] Gates foram considerados.

## Conclusão

Este prompt transforma intenção em execução controlada.
