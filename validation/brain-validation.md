# Validação de Brains

## Objetivo

Auditar se os brains da CEIP respondem às perguntas certas e interagem corretamente com agents e engines.

## Perguntas de validação

- Cada brain tem missão?
- Cada brain tem responsabilidades?
- Cada brain informa quando é ativado?
- Cada brain declara perguntas que responde?
- Cada brain lista documentos consultados?
- Cada brain informa agentes com que interage?
- Cada brain define decisões que pode e não pode tomar?
- Cada brain inclui exemplos?
- Cada brain tem checklist?
- O Engineering Intelligence Core coordena o fluxo completo?

## Checklist

- [ ] Todos os brains obrigatórios existem.
- [ ] Nenhum brain assume tecnologia específica.
- [ ] Limites de decisão estão claros.
- [ ] Interações com agentes são explícitas.
- [ ] O fluxo Mermaid do Core existe.

## Conclusão

Brain válido reduz ambiguidade cognitiva para agentes de IA e humanos.
