# Template de Validação em Projeto Real

## Objetivo

Padronizar o registro de experimentos do framework em projetos reais.

## Contexto

Cada piloto deve gerar evidência sobre o funcionamento do framework, não apenas opinião geral.

## Template

```markdown

- Não inventar regra de negócio.
- Registrar lacunas.
```
