# Performance Score

## Objetivo

Pontuar adequação de performance com base em metas, medições e impacto operacional.

## Contexto

Performance deve ser avaliada contra objetivo real, volume esperado e restrições do ambiente.

## Critérios

| Critério | Peso sugerido |
| --- | --- |
| Meta de performance definida | 20 |
| Medição antes/depois | 25 |
| Consultas e processamento críticos | 20 |
| Uso de recursos | 15 |
| Monitoramento | 10 |
| Trade-offs registrados | 10 |

## Interpretação

- 90-100: excelente.
- 80-89: bom.
- 70-79: aceitável com ressalvas.
- 60-69: fraco.
- Abaixo de 60: bloqueado.

## Checklist

- [ ] Objetivo foi definido.
- [ ] Medição foi feita.
- [ ] Gargalos foram tratados por prioridade.
- [ ] Monitoramento foi definido.
- [ ] Não houve otimização especulativa.

## Conclusão

Performance aceita precisa ser mensurada e contextual.
