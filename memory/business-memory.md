# Business Memory

## Objetivo

Registrar aprendizados de negócio reutilizáveis em projetos CloudSix.

## Conteúdo permitido

- Processos recorrentes.
- Regras validadas de forma genérica.
- Termos de domínio.
- Critérios de aceite recorrentes.

## Limites

- Não registrar dados pessoais ou comerciais sensíveis.
- Não transformar hipótese em regra.
- Não generalizar contexto específico sem validação.

## Checklist

- [ ] A regra foi confirmada.
- [ ] O contexto foi descrito.
- [ ] O registro é reutilizável.
- [ ] Não há dado sensível.

## Conclusão

Memória de negócio deve preservar aprendizado sem inventar domínio.
