# Decision Memory

## Objetivo

Registrar histórico resumido de decisões recorrentes e seus motivos.

## Conteúdo permitido

- Decisão.
- Contexto.
- Alternativas rejeitadas.
- Critério de escolha.
- Link para ADR, RFC ou policy.

## Limites

- Não substituir ADR obrigatório.
- Não registrar decisão sem responsável ou data.

## Checklist

- [ ] A decisão tem contexto.
- [ ] Alternativas foram consideradas.
- [ ] Há responsável ou origem.
- [ ] Documento formal foi ligado quando aplicável.

## Conclusão

Memória de decisão ajuda agentes a não reabrirem escolhas sem novo contexto.
