# Client Memory

## Objetivo

Registrar aprendizados gerais sobre perfis de cliente, comunicação e restrições recorrentes.

## Conteúdo permitido

- Preferências operacionais genéricas.
- Restrições técnicas ou regulatórias.
- Padrões de comunicação.
- Cuidados de implantação.

## Limites

- Não registrar dado pessoal.
- Não registrar informação comercial sensível sem necessidade.
- Não registrar julgamento subjetivo sobre pessoas.

## Checklist

- [ ] Informação é necessária para execução futura.
- [ ] Informação está anonimizada quando possível.
- [ ] Não há dado sensível desnecessário.
- [ ] Origem foi registrada.

## Conclusão

Memória de cliente deve apoiar atendimento sem invadir privacidade.
