# Lesson Learned Memory

## Objetivo

Registrar lições aprendidas em projetos, incidentes, releases e revisões.

## Conteúdo permitido

- Situação.
- O que funcionou.
- O que não funcionou.
- Mudança recomendada.
- Documento que deve ser atualizado.

## Limites

- Evitar generalizações sem evidência.
- Proteger dados sensíveis.

## Checklist

- [ ] A lição tem contexto.
- [ ] A recomendação é prática.
- [ ] Documento relacionado foi apontado.
- [ ] A memória tem validade futura.

## Conclusão

Lição aprendida só tem valor quando muda uma prática.
