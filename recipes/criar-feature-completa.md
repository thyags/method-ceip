# Recipe: Criar Feature Completa

## Objetivo

Executar feature de ponta a ponta com negócio, produto, arquitetura, implementação, testes e documentação.

## Entradas

Problema, usuários, critérios de aceite, restrições, risco, stack existente e dependências.

## Passos

1. Validar problema e escopo.
2. Classificar risco no Policy Engine.
3. Selecionar agentes no Orchestrator.
4. Produzir especificação e plano técnico.
5. Implementar em etapas pequenas.
6. Executar reviews e quality gates.
7. Atualizar documentação, ADR, knowledge ou memory quando aplicável.

## Saídas

Feature validada, evidências, documentação e decisão de aprovação.

## Checklist

- [ ] Escopo foi fechado.
- [ ] Risco foi classificado.
- [ ] Agentes foram acionados.
- [ ] Gates foram aplicados.
- [ ] Evidências foram registradas.

## Conclusão

Feature completa é entrega validada, não apenas código pronto.
