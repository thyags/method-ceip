# Recipe: Criar Integração Externa

## Objetivo

Orientar integrações com APIs, webhooks, importações, ERPs, CRMs, pagamentos ou marketplaces.

## Entradas

Contrato externo, autenticação, limites, payloads, erros, volume, SLA e dados sensíveis.

## Passos

1. Acionar API Integration Engineer, Security Engineer e Backend Engineer.
2. Documentar contrato e autenticação.
3. Definir idempotência, retry, timeout e fallback.
4. Mapear erros e observabilidade.
5. Aplicar gates de API, segurança, backend, QA, performance e documentação.

## Saídas

Contrato de integração, plano de erro, plano de teste, monitoramento e documentação.

## Checklist

- [ ] Contrato foi entendido.
- [ ] Autenticação foi validada.
- [ ] Idempotência foi considerada.
- [ ] Erros e retries foram definidos.

## Conclusão

Integração externa precisa falhar de forma controlada.
