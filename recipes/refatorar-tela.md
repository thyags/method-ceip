# Recipe: Refatorar Tela

## Objetivo

Melhorar tela existente sem alterar comportamento esperado indevidamente.

## Entradas

Tela atual, problemas observados, métricas, feedback de usuário, componentes existentes e restrições.

## Passos

1. Registrar comportamento atual.
2. Mapear estados e fluxo.
3. Acionar Frontend UX Specialist, UI Designer e QA Engineer.
4. Refatorar em etapas pequenas.
5. Validar responsividade, acessibilidade e regressão.

## Saídas

Tela refatorada, evidências visuais, testes e documentação de mudança.

## Checklist

- [ ] Comportamento atual foi preservado ou mudança foi aprovada.
- [ ] Estados foram cobertos.
- [ ] Layout foi validado.
- [ ] Regressão foi testada.

## Conclusão

Refatoração de tela deve melhorar uso sem surpreender operação.
