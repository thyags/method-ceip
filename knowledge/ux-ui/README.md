# Knowledge UX/UI

## Objetivo

Reunir conhecimento sobre experiência, interface, acessibilidade, responsividade e consistência visual.

## Conteúdos esperados

- Fluxos de usuário.
- Padrões de componentes.
- Estados de interface.
- Acessibilidade.
- Responsividade.

## Checklist

- [ ] O aprendizado melhora uso real.
- [ ] Há exemplo visual ou comportamento esperado quando útil.
- [ ] Estados de erro, vazio e carregamento foram considerados.
- [ ] O conteúdo respeita padrões locais de cada projeto.

## Conclusão

UX/UI na CEIP deve ajudar operação real e reduzir fricção.
